ComponentOne Invexplorer

This application was implemented using TypeScript (typed, error-checked JavaScript), KnockoutJS (data binding, MVVM support), jQueryUI (auto-complete control used to select new symbols), jQuery/Globalize (formatting numbers and dates), ComponentOne Wijmo (chart control), and Google Finance (inspiration).

The data displayed is obtained from Yahoo finance services. Yahoo services are not free; they are being used here for demonstration purposes only.
