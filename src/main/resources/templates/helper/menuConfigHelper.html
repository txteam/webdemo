配置文件：
		src/main/resources/context/menu_config.xml
		classpath:/context/menu_config.xml
	
	当前项目中仅存在主菜单：
	所以这里仅对主菜单配置进行说明：
		菜单项：
			menu 
				id="xxx" 
				text="菜单管理"
				href="/mainframe/toQueryAllMenuItemTreeList.action" 
				auths="menu_config"  or roles="..."  多个权限或角色，可以使用逗号分隔
				type="tab"
				...=...
				
				选填
				icon="..."
				tips="..."
				可动态扩展


	

		

			id
			菜单项唯一键,不允许重复，如果重复启动期间将会出现异常
		

		

			text
			菜单名称，直接关系到菜单中显示的内容
		

		

			type
			
				菜单弹出目标：合法值（ tab,nav,event,dialog...可扩展）

				tab:将在页面center中显示href的链接
				dialog:将打开一个id=menu.id的dialog对象，如果该对象已经存在，则显示该dialog.根据menu.modal决定弹出的dialog是否为模态
				event:出发一个menu.eventType的全局事件
					
		

		

			href
			菜单名称，直接关系到菜单中显示的内容
		

		

			tips
			鼠标悬停菜单时显示信息（暂未进行支持）
		

		

			auths
			菜单项，所需权限，如果菜单项不需要权限则不设置该值即可。如果父级菜单需要权限，对应菜单项目，则默认为应该具有父级菜单权限才可以显示
		

		

			roles
			菜单项，所需权限，如果菜单项不需要权限则不设置该值即可。如果父级菜单需要权限，对应菜单项目，则默认为应该具有父级菜单权限才可以显示
		

		

			valid
			菜单项是否有效(暂时未进行支持)
		

		

			visible
			菜单项是否可见(暂时未进行支持)
		

		

			selectRefresh
			如果target为mainTabs时：点选菜单时是否需要刷新tabs(暂时未进行支持)
		

		

			openNewEveryTime
			如果target为mainTabs时：点选菜单时是否每次都打开一个新的tab(暂时未进行支持)
		

		

			width
			如果target为mainTabs时：点选菜单时是否每次都打开一个新的tab(暂时未进行支持)
		

		

			height
			如果target为mainTabs时：点选菜单时是否每次都打开一个新的tab(暂时未进行支持)
